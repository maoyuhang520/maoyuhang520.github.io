    一.HTML简介

    1.HTML概念：

    HTML是超文本标记语言

    标记语言是一套标记标签，HTML使用标记标签描述网页

    2.HTML标签：

    尖括号包围

    开放标签和闭合标签

    html
    3.HTML文档：

    HTML文档即web页面，包含HTML tag和纯text

    Web阅览器(chrome，Internet Explorer，Firefox，Safari )可读取HTML文档，使用标签解释页面内容

    html
