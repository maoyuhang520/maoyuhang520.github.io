ALDS1_3_B:队列

有n队列中的进程。每个进程都有名称。i和时间i。循环调度按顺序处理进程。循环调度程序为每个进程提供一个量子(一个时隙)如果到那时还没有完成，就会中断这个过程。进程被恢复并移动到队列的末尾，然后调度程序处理队列中的下一个进程。

例如，我们有以下的队列，其量程为100 ms。

Code

首先，进程A被处理100 ms，然后进程被移动到队列的末尾，剩下的时间(50 Ms)。

Code

接下来，进程B的处理时间为80 ms。该过程以180 ms的时间戳完成，并从队列中删除。

Code

您的任务是编写一个模拟循环调度的程序。.

输入

n q
名字1时间1
名字2时间2
…
名字n时间n

在第一行中，进程的数量。n以及量子q都是用一个单独的空格隔开的。

在以下几个方面n的行、名称和时间。n给出了过程。名字i和时间i由一个空格隔开。

输出量

对于每个进程，按顺序打印其名称和进程完成的时间。

约束

  • 1≤n≤100000
  • 1≤q≤1000
  • 1≤时间i≤50000
  • 1≤长度名字i≤10
  • 1≤和时间i≤1000000
