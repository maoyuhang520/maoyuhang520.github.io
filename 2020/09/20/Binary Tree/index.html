    ALDS1_7_B: 二叉树的表达

    二叉树

    根二叉树是一个根节点的树，每个节点最多有两个子节点。

    您的任务是编写一个读取根二叉树的程序。T并打印每个节点的下列信息u的T:

    • 节点IDu
    • 父母u
    • 兄弟姐妹u
    • 儿童人数u
    • 深u
    • 高度u
    • 节点类型(根、内节点或叶)

    如果两个节点具有相同的父节点，则它们是兄弟姐妹。在这里，如果u和v有同样的父母，我们说u是兄弟姐妹v(反之亦然)。

    树中节点的高度是从节点到叶子的最长简单向下路径上的边缘数。

    在这里，给定的二叉树由n节点和evey节点有一个唯一的ID，从0到n-1.

    输入

    输入的第一行包括一个整数。n，树的节点数。

    在下一个n行，每个节点的信息以下列格式提供：

    ID左右

    ID是节点ID，左边是左子的ID和正确的是正确孩子的ID。如果节点没有左(右)子节点，则左边(正确的)由-1.

    输出量

    以下列格式打印每个节点的信息：

    节点 ID: 父母=p兄弟姐妹=s，度=德格，深度=副署长，高度=h, 类型

    p是其父级的ID。如果节点没有父节点，则打印 -1.

    s是它兄弟姐妹的ID。如果节点没有同级，则打印 -1.

    德格, 副署长和h分别是子节点的个数、节点的深度和高度。

    类型是由字符串(根部, 内节点或叶。如果根可以视为叶或内部节点，则打印根部.

    请遵循以下示例输出中提供的格式。

    约束

    • 1≤n≤25
