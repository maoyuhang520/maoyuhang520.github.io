    Rooted Tree

    ALDS1_7_A: 有根树的表达

    图G = (V, E)是一种数据结构，其中V是一个有限的顶点集，并且E上的二元关系V由一组边缘表示。图1示出了图(或多个图)的示例。


    图1

    自由树是连接的、无环的、无向图。根树是一棵自由树，其中一个顶点被区分开。根树的顶点称为“节点”。

    您的任务是编写一个程序，报告每个节点的下列信息u一棵有根的树T:

    • 节点IDu
    • 父母u
    • 深u
    • 节点类型(根、内节点或叶)
    • 孩子的名单u

    如果路径上的最后一条边沿从根目录r一棵树T到节点x是(p, x)，那么p是父母的x，和x是儿童的p。根是T没有父母。

    没有子节点的节点是外部节点或叶。非叶节点是内节点

    节点的子节点数。x在一棵有根的树上T被称为度的x.

    根路径的长度。r到节点x是深度的x在……里面T.

    在这里，给定的树由n节点和evey节点有一个唯一的ID，从0到n-1.

    图2示出了以圆圈(节点)中的数字表示每个节点的ID的根树的示例。该示例对应于第一示例输入。


    图2

    输入

    输入的第一行包括一个整数。n，树的节点数。

    在下一个n行，每个节点的信息u以下列格式提供：

    ID k c1 c2 … ck

    哪里ID的节点ID。u, k是.的程度u, c1 … ck是第一个节点ID.k.的第二个孩子u。如果节点没有子节点，则k是0。

    输出量

    以ID排序的格式打印每个节点的信息：

    节点 ID: 父母=p，深度=d, 类型, [c1…ck]

    p是其父级的ID。如果节点没有父节点，则打印 -1.

    d节点的深度。

    类型是由字符串(根部, 内节点或叶)。如果根可以视为叶或内部节点，则打印根部.

    c1…ck作为有序树的子列表。

    请遵循以下示例输出中提供的格式。

    约束

    • 1≤n≤100000
