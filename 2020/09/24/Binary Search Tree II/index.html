    Binary Search Tree II

    ALDS1_8_B: 二叉搜索树的插入 II

    编写一个程序，通过将查找操作添加到A中，对二进制搜索树$T$执行以下操作：二进制搜索树I。

    • 插入$k$：将包含$k$as key的节点插入$T$。
    • 发现$k$：报告$T$是否有包含$k$的节点。
    • 列印分别通过无序树遍历和预排序树遍历打印二进制搜索树的键。

    输入

    在第一行中，给出了$m$的操作数。在以下$m$行中，由插入$k$，发现$k$或列印都是给你的。

    输出量

    每人发现$k$运算，打印“是“如果$T$有包含$k$的节点，”不“”如果没有。

    另外，每个列印操作，分别在一行中打印顺序树遍历和预排序树遍历获得的密钥列表。放置空格字符在每个键之前.

    约束

    • 行动次数$$500 000$
    • 打印操作数$\leq 10$。
    • $-2,000,000\leq Key\leq 2,000,000,000,000,000
    • 如果使用上述伪代码，二叉树的高度不超过100。
    • 二进制搜索树中的键都是不同的。
